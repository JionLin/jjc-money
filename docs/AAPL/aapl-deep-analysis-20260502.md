## 三、估值快扫 · §2.1 PE-Based Valuation

```
Step 1 — 增长档位识别
  FY2026E EPS增速 ~14-16% → "Mature-Growth"过渡带
  FY2027E EPS增速 ~10%    → "Mature"档
  可接受Forward PE区间：15-22x（按Mature），18-27x（按Growth）
  → 综合给 18-25x 为合理中枢

Step 2 — PEG交叉验证
  Forward PE ~31.2x / EPS增速 ~14-16%
  PEG = 31.2 / 15 ≈ 2.08
  → 🔴 PEG > 2.0 = 红旗警示区

Step 3 — 现金流健全检查
  FY2026E FCF ~$1,360-1,370亿
  FCF margin ~30%+ → ✅ 极高质量
  FCF > Net Income → ✅ 现金流充分覆盖利润
  FCF Yield ~3.3%（基于$4.1万亿市值）→ ⚠️ 偏低

Step 4 — 历史自我比较
  5年Forward PE中位数 ~27x
  当前 ~31.2x → 位于历史上沿75%分位
  → ⚠️ 偏贵，但未到极端

Step 5 — 综合判断
  "估值合理偏贵。FCF质量顶级，但PEG > 2.0敲响警钟。
   iPhone 17兑现后，市场已经在定价FY2027的减速。"
```

### PEG 信号灯

```
PEG ≈ 2.08 → 🔴 红旗警示 → 当前价位不宜追高
```

> **但要注意**：苹果的PEG常年偏高（品牌溢价+回购溢价+生态锁定溢价），单看PEG容易误杀。**关键是FCF margin 30%+和年化$1,370亿的自由现金流，这台印钞机没有任何停转的迹象**。

## 五、赛道归类 · Model 5 粪坑检测

```
□ 赛道处于 >12个月下行趋势？ → ❌ 否。消费科技趋势稳健
□ 基本面（营收/FCF）逐季恶化？ → ❌ 否。Q2营收+17%，EPS+22%
□ 同业违约/退市？ → ❌ 否。苹果是全球市值第一梯队
□ 监管环境持续收紧无缓解？ → ⚠️ 部分。App Store反垄断仍在推进
```

**赛道定位**：

在三账户体系中，苹果是**进取型账户的"压舱石"**，不是主攻手。

| 维度 | 苹果角色 | vs. 英伟达对比 |
|------|---------|--------------|
| 进取型账户占比 | ~4.5-5.5% | ~46% |
| 波动率 | 低-中 | 高 |
| 增长弹性 | 中等（14-16%） | 极高（40%+） |
| 防守能力 | 强（FCF+回购+品牌） | 弱（高波动） |
| 定位 | 底仓标配 | 核心重仓 |

## 九、风险与反方 · 什么会让这个论点出错？

### 9.1 利多因素（支撑当前估值的力量）

| 因素 | 强度 | 说明 |
|------|:----:|------|
| iPhone 17超级周期已验证 | ★★★★☆ | Q2 iPhone +21.7%，不再是预期 |
| Services飞轮加速 | ★★★★★ | $310亿/季新高，边际成本趋零 |
| 中国市场回暖 | ★★★★☆ | +28.1%，超预期 |
| 回购机器 | ★★★★★ | 新增$1,000亿授权，EPS增厚确定性高 |
| Apple Intelligence分发 | ★★★☆☆ | 25亿设备是AI落地最大终端入口 |

### 9.2 利空因素（可能打破论点的风险）

| 风险 | 概率 | 潜在影响 |
|------|:----:|---------|
| **FY2027 EPS增速放缓至~10%** | 高 | PE从31x压缩至25-27x → 股价$230-$250 |
| **关税/贸易摩擦升级** | 中 | 供应链成本上升，中国市场再度承压 |
| **芯片/内存涨价侵蚀毛利** | 中 | 作者预判"一二季度财报开始呈现" |
| **App Store反垄断** | 中-低 | 若抽成降至15%以下→Services利润重构 |
| **AI功能差异化不足** | 中 | Apple Intelligence vs. 安卓AI对手 |
| **iPhone 18/19周期验证** | 待定 | 超级周期不可能年年有 |

### 9.3 最大风险一句话

> **苹果最大的风险不是基本面恶化，而是"增长减速+估值压缩"的双杀**。FY2027 EPS增速如果掉到10%以下，市场可能把Forward PE从31x压到25x以下——那就是$210的事。去年Meta和微软都经历过类似的"增速不达预期→股价暴跌"，苹果不是免疫体。
