## 三、估值快扫（§2.1 PE估值框架）

### Step 1 — 增长体制判定

| 指标 | 数值 | 判定 |
|------|:----:|:----:|
| 2026E EPS增速（整体） | 13-17% | **Mature → Growth过渡带** |
| 信息技术板块增速 | 40%+ | **Hyper-growth** |
| Trailing PE | 33.6-36.8x | 历史高位 |
| Forward PE | ~23x | |
| 5年均值 Forward PE | ~27x | |
| 10年均值 Forward PE | ~23x | |

按金渐成估值框架，整体EPS增速13-17%：
- Mature (10-20%) → 合理 Forward PE: 12-18x
- Growth (20-30%) → 合理 Forward PE: 18-27x

但这里有个陷阱：**纳指100的增速是严重分化的。** 科技板块40%+的增速在拉高平均值，而其他板块的增速可能只有个位数。

当前 Forward PE ~23x → **如果看整体增速13-17%，PE偏贵；如果看科技板块40%+，PE合理。**

这就是纳指100估值的核心矛盾——你得先想清楚，你买的到底是"100家公司的平均值"，还是"七巨头的打包套餐"。

### Step 2 — PEG交叉校验

```
PEG = 23 / 15 ≈ 1.53（按整体增速15%）
PEG = 23 / 40 ≈ 0.58（按科技板块增速40%+）
```

🟠 **按整体增速：PEG 1.53 → 合理偏贵，不追高**
🟢 **按科技板块：PEG 0.58 → 潜在低估**

真相在中间。纳指100的PEG取决于你怎么看待AI CapEx→收入兑现这个故事。信则"便宜"，不信则"贵"。

> **"单一指标很危险。至少用 Forward PE + PEG + FCF 交叉验证。没有一个数字能说明全部。"**

### Step 3 — 现金流健全性检查

纳指100作为ETF本身没有FCF，但我们看其核心持仓：

| 公司 | FCF状态 | CapEx趋势 |
|------|:-------:|:---------:|
| NVDA | ✅ FCF margin 极高 | CapEx适中 |
| AAPL | ✅ FCF > Net Income | CapEx稳定 |
| MSFT | ⚠️ FCF受Azure CapEx挤压 | CapEx暴增 |
| GOOGL | ⚠️ FCF受AI CapEx挤压 | CapEx暴增 |
| AMZN | ⚠️ FCF受AWS CapEx挤压 | CapEx暴增 |
| META | ⚠️ FCF受AI CapEx挤压 | CapEx暴增 |

**结论**：七巨头中，除了NVDA和AAPL，其余四家的FCF都在被巨额AI CapEx吃掉。这不代表它们不赚钱，但意味着"纸面利润"和"口袋里的钱"之间有差距。

> "FCF < Net Income → DIG DEEPER。" 当前不至于会计造假，但CapEx这台吞金兽必须被盯住。

### Step 4 — 历史自身对比

| 维度 | 当前 | 10年均值 | 位置 |
|------|:----:|:------:|:----:|
| Trailing PE | 33-37x | ~27x | **显著高于均值** |
| Forward PE | ~23x | ~23x | **持平均值** |
| 52周高点 | 27,800+ | — | **接近新高** |
| 52周低点 | 19,621 | — | 距低点+41% |

**结论**：Trailing PE明显偏高，但Forward PE因强劲盈利预期被拉回到了10年均值附近。关键变量是"盈利预期能不能兑现"。

### 估值总判定

```
🟡 估值合理偏贵 — Forward PE持平历史均值，但Trailing PE显著偏高。
估值能否维持取决于AI CapEx→收入兑现。
当前不是追高的时候，也不构成卖出信号。
持有为主，等回调再加。
```

## 六、纳指100的结构性风险（Model 5 · 粪坑检测 → 不适用，但隐患比标普500更大）

纳指100不是粪坑——恰恰相反，它是过去三年全球表现最好的主流指数之一。但"表现好"不等于"没风险"。

### 🔴 风险1：集中度比标普500更极端

七巨头占纳指100权重约40%，前十大约47%。

- NVDA跌20% → 纳指跌 ~2%
- 七巨头集体回调15% → 纳指可能跌 **6-8%**（比标普影响更大）
- **你以为买了100家公司，实际上近一半的钱押在10家公司上**

**这不是"分散投资"，这是"穿着100件衣服的集中押注"。**

### 🔴 风险2：科技赛道单一暴露

标普500至少还有金融（12%）、工业（8.8%）、医疗（8.5%）来对冲。纳指100？科技+通信+可选消费占80%以上。

如果AI叙事出现裂缝 → 标普还有其他板块撑着 → 纳指100无处躲藏。

> "干嘛非要在一个趋势下行的板块里找黄金呢？" 反过来说，当趋势向上时，集中在单一赛道回报最高；但转向时，也最痛。

### 🟡 风险3：AI CapEx→收入兑现的"信仰测试"

作者原话：**"AI前期巨大的资本支出，需要让大多数企业的赚钱预期成功落地，不然会动摇市场信心。"**

纳指100的核心持仓（MSFT/GOOGL/AMZN/META）2026年CapEx合计数千亿美元。如果下半年财报显示"花了钱但没赚到钱"，纳指100首当其冲——因为这些公司在纳指100里的权重比标普500里更高。

### 🟡 风险4：6月-9月的时间窗口

作者年度展望中明确警告的风险窗口对纳指100的影响尤其大：

- **6月降息预期落空** → 成长股估值最先被杀 → 纳指跌幅 > 标普
- **中期选举政策不确定性** → 科技监管风险
- **报税季资金抽水** → 高估值板块最先被抽血

### 🟢 利好因素

- AI趋势依然处于"适度泡沫的早期阶段，远未到破裂时"
- 科技板块EPS增速40%+，领跑全市场
- 降息周期对成长股估值最为友好
- Q1财报季整体超预期，基本面验证中
- SpaceX/OpenAI等巨头IPO可能推动科技板块情绪

## 十二、财报专业名词入门指南

> 📖 看不懂下面这些词，就看不懂上面的分析。花3分钟搞明白，受益一辈子。

| 英文缩写 | 中文名 | 一句话解释 |
|---------|--------|-----------| 
| **Nasdaq-100** | 纳斯达克100指数 | 纳斯达克交易所市值最大的100家非金融公司组成的指数 |
| **QQQ** | 纳指100 ETF | 追踪纳斯达克100指数的最大ETF基金，买QQQ≈买纳指100 |
| **PE (Price-to-Earnings)** | 市盈率 | 股价÷每股盈利。PE=30意味着你花30块买1块钱的年利润 |
| **Forward PE** | 前瞻市盈率 | 用未来12个月预估盈利算的PE，比Trailing PE更有前瞻性 |
| **Trailing PE** | 滚动市盈率 | 用过去12个月实际盈利算的PE，反映历史 |
| **EPS (Earnings Per Share)** | 每股盈利 | 公司利润÷总股数。EPS越高，赚钱能力越强 |
| **PEG** | 市盈增长比率 | PE÷盈利增速。PEG≈1合理，<0.8便宜，>1.5偏贵 |
| **FCF (Free Cash Flow)** | 自由现金流 | 经营赚的钱减去必要投资后剩下的"真金白银" |
| **CapEx (Capital Expenditure)** | 资本支出 | 企业花在建厂房、买设备、建数据中心上的钱 |
| **VIX** | 波动率指数 | 市场恐慌程度的温度计。<20平静，>30恐慌，>40极度恐慌 |
| **YTD (Year-to-Date)** | 年初至今 | 从1月1日到现在的累计涨跌幅 |
| **ETF** | 交易所交易基金 | 像股票一样交易的基金，一篮子股票打包卖给你 |
| **费率 (Expense Ratio)** | 管理费率 | 基金每年收取的管理费占比。QQQ 0.20%，即每投$10,000年收$20 |
| **股息率 (Dividend Yield)** | 分红收益率 | 年度分红÷股价。QQQ约0.42%，不以分红见长 |
| **52周高/低** | 52周最高/最低价 | 过去一年的价格极值，衡量当前价格在历史区间中的位置 |
| **市值 (Market Cap)** | 市场资本化 | 股价×总股数，代表市场认为这家公司值多少钱 |
