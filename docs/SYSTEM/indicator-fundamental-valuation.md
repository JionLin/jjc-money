# 金渐成体系指标详解：基本面估值指标 (Fundamental & Valuation Indicators)

> **所属心智模型**：PE-Based Valuation Quick-Scan (估值速查模型)
> **核心原则**：单个指标是危险的，必须至少将 Forward-PE + PEG + FCF 交叉验证。

金渐成体系不买“讲故事”的公司，而是寻找业绩有坚实支撑、且估值（种子）价格合理的标的。核心是通过以下指标确定**安全边际**。

## 1. 远期市盈率 (Forward PE) 与增长率匹配
不能孤立地看 PE 是高还是低，必须将其与公司的预期盈利增长率（EPS Growth）相匹配。
- **判断逻辑**：高增长享受高估值，低增长只能给低估值。
- **实战参考带**：
  - **超高速增长** (EPS 增速 >40%)：可接受 Forward-PE 40-55x
  - **高速增长** (EPS 增速 30-40%)：可接受 Forward-PE 27-40x
  - **稳定增长** (EPS 增速 20-30%)：可接受 Forward-PE 18-27x
  - **成熟期** (EPS 增速 10-20%)：可接受 Forward-PE 12-18x
  - **价值/衰退期** (EPS 增速 <10%)：可接受 Forward-PE 8-14x
- **历史纵向对比**：当前的 Forward-PE 处于该公司过去 5 年的什么位置？（Top 10% 则偏贵考虑减仓，Bottom 10% 则是潜在机会）。

## 2. PEG 信号灯 (PEG = Forward-PE / 预期 EPS 增速)
这是体系内最核心、最直观的估值速查器，用于快速过滤标的。
- **判断逻辑**：衡量估值与增长潜力的性价比。
- **信号灯速查表**：
  - 🟢 **PEG < 0.8**：潜在低估，安全边际高，可结合恐慌情绪分批建仓。
  - 🟡 **PEG 0.8 - 1.2**：合理估值区间，适合持有或利用底仓做 T。
  - 🟠 **PEG 1.2 - 1.5**：合理但偏贵，不建议追高，观望为主。
  - 🔴 **PEG > 2.0**：红旗警示，估值严重透支，除非是处于全新范式爆发极早期，否则坚决回避或逢高减仓。

## 3. 自由现金流 (Free Cash Flow, FCF)
这是验证盈利质量的终极试金石。
- **判断逻辑**：“有利润没现金”的公司极可能存在会计粉饰，只有 FCF 才是真金白银。
- **实战应用**：
  - **FCF Margin > 20%**：被视为高品质盈利的标志（如微软、苹果等防守/进取核心资产）。
  - **FCF > 净利润**：证明公司的盈利是由实打实的现金支撑的。
  - **反面预警**：如果 FCF < 净利润，且持续恶化，必须深挖其财务报表，往往是暴雷前兆。
