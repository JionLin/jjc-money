# 金渐成体系指标详解：宏观与情绪指标 (Macro & Sentiment Indicators)

> **所属心智模型**：Model 1 (顺势 · Ride the Trend) / Model 2 (激流缓退 · Gradual Exit in Rapid Flow)
> **核心公理**：“从来不是人赚钱，是钱在找寻人。” / “买在无人问津处，卖在人声鼎沸时。”

在“金渐成”投资体系中，宏观环境和市场情绪是决定个股和资产配置大方向的“风”。在看任何具体的公司财报或技术图形之前，必须先判定当前的宏观周期和情绪水温。

## 1. 美联储利率周期 (Fed Funds Rate)
这是宏观趋势的“总开关”。
- **判断逻辑**：货币政策的松紧直接决定了市场的流动性充裕程度。
- **实战应用**：
  - **降息周期/宽松预期**：利好进攻型赛道（如 AI、半导体）以及对利率敏感的中小盘股（美国周期/中小盘补涨）。
  - **加息周期/高息维持**：压制高估值科技股，此时应侧重防守，或利用高息环境锁定美债（如 TLT、短债）作为防守垫。

## 2. 汇率指标 (主要是 USD/CNY)
美元是全球资金流动的基石，汇率是跨市场配置的核心参考。
- **判断逻辑**：美元强弱决定了资金是回流美国还是溢出到新兴市场。
- **实战应用**：
  - **关键节点**：人民币汇率 7.25 - 7.35 是历史操作中反复提及的观察区间。
  - **资金路由**：当美元极强时，坚守美股核心资产；当美联储降息预期明确、美元走弱时，才考虑通过港股等渠道布局中国资产的反弹（前提是基本面配合）。

## 3. 资本支出 (CapEx) 趋势
特别针对科技和 AI 产业链，这是验证“宏观叙事”是否能落地的宏观级别指标。
- **判断逻辑**：下游大厂（如微软、谷歌、亚马逊、Meta）的资本支出，就是上游硬件厂（如英伟达、台积电）的营收。
- **实战应用**：
  - 只要北美云巨头的 CapEx 保持高增长（如 2024-2026年），AI 硬件产业链的业绩确定性就极高。
  - **预警信号**：如果大厂 CapEx 增速放缓，或云业务收入未能兑现（无法消化 CapEx），则是整个 AI 赛道趋势反转的信号。

## 4. 市场情绪指标 (Sentiment)
用于判定何时进场、何时离场。
- **判断逻辑**：情绪总是在极度恐慌和极度贪婪之间摇摆。
- **实战应用**：
  - **无人问津（恐慌）**：伴随 VIX 飙升、负面新闻满天飞。这是**启动 2-3-3-2 建仓程序**的信号。
  - **人声鼎沸（鼎沸）**：媒体集体唱多、估值突破天际（如 NVDA 突破特定市值大关）。这是**启动分批减仓、提取本金转为“负成本”**的信号。
