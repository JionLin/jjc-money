# 金渐成体系指标详解：价格行动与技术支撑 (Price Action & Technical Support)

> **所属心智模型**：2-3-3-2 法则 / 金字塔补仓法
> **核心思路**：不预测绝对底部，而是在高确定性的支撑区域分步收集廉价筹码。

在确认了宏观趋势向上、估值合理且非“粪坑”后，具体的买卖点设定并不依赖复杂的技术指标（如缠论、波浪），而是基于常识、心理关口和历史筹码分布。

## 1. 支撑位/买点设定逻辑 (Support-Zone Setting)
在战术接货时，挂单位置的选取原则：
- **前期恐慌底 (Prior Panic Low)**：寻找该资产上一次因为宏观情绪（非基本面暴雷）导致的巨幅杀跌低点。这里通常是长线资金认可的“铁底”。
- **巨量成交密集区 (High-volume Congestion Zone)**：历史上换手极高、筹码沉淀充分的价格区间。
- **整数关口与前高前低转换**：利用市场的心理锚点（如 $100, $150 等整数位），以及曾经的强阻力位突破后转化为强支撑的位置。
- **微调规则**：如果价格距离预设支撑位仅差 <1%，且买入逻辑未变，允许进行极小幅度的向上微调介入，但**绝不整体上移总风险预算**。

## 2. 金字塔补仓倍数 (Pyramid Multiplier)
结合支撑位，制定阶梯式的挂单数量。越往下跌，买入量越大。
- **标准倍数**：1 → 1.5 → 2 → 3（价格越低，单笔买入股数越多）。
- **细化战术挂单**：1 → 1 → 1.5 → 1.5 → 2 → 2.2/2.5（常用于主线极其明确，借大盘回调时的网格接货。2.5 倍的最重仓位只留给极端的恐慌深水区）。

## 3. 减仓/卖点设定逻辑 (Sell-Point Identification)
卖出往往是为了实现“负成本”，而非彻底清仓。
- **触及预设阻力区**：达到建仓时设定的目标价或历史高点压力位。
- **估值进入极度高估区**：如 Forward PE 进入历史 Top 10% 分位。
- **情绪鼎沸**：执行 2-3-3-2 的逆向操作（20% 试探减仓 → 30% 鼎沸减仓 → 30% 目标减仓 → 20% 留作负成本底仓）。

## 4. 日历与周期波动风险 (Calendar Risk)
金渐成体系在交易节奏上会充分考虑特定的时间节点带来的波动。
- **美国报税季效应**：特别是 4 月中旬的年度报税期，以及 9 月的季度纳税期，市场往往面临短期的流动性抽离，容易砸出“黄金坑”。
- **夏秋高波动窗口**：6 月初到 9 月初，若叠加宏观事件（如降息预期落空、大选/中期选举），通常是 VIX 容易飙升、适合执行恐慌买入的阶段。
