### 第三步：MSFT 的处理 — 暂不动，等高位减

```
当前 MSFT 52 股 × $414 ≈ $21,538 → 占比 ~39%
目标 Phase 1：35-38%
目标 Phase 2：30-32%

→ 当前不需要主动减 MSFT
→ 等 MSFT 涨到 $500+ 再开始试探性减仓
→ 减仓节奏沿用你现有计划：
   $500-520 → 卖 5-8 股

   $600+    → 继续视情况分批
```

> **为什么不现在卖 MSFT 去换 NVDA？**
> 因为 MSFT $414 处于作者"可口饭菜区"的上沿（$400 以下才是甜蜜区），现在卖 = 低位割肉。
> 而 NVDA $200 处于作者"持有观望区"（$187-$240），现在买 = 偏高追入。
> **两头都不合适 = 等。**

## 五、操作纪律（Discipline）

### ✅ 该做的

1. **UNH 先走** → 它不是未来核心，回笼资金进 NVDA 弹药池
2. **NVO 利用反弹清掉** → 备胎完成使命就走

5. **保留至少 20% 现金** → 现金像氧气，没有时才知道贵

### ❌ 不该做的

2. **不要为了让 NVDA 追上 MSFT 而低价卖 MSFT** — 割下面补上面 = 两头亏
3. **不要一次性把所有弹药打在 NVDA 一个价位** — 分批！分批！分批！
4. **不要为了"向作者看齐"而忽略自己的资金规模差异** — 他的 46% 是三年种出来的，不是种子期就打满的
5. **不要把现金全打光** — 你暂时没有新增资金流入，现金就是你的唯一后备军
