## 一、个股分析（最常用）

### 基础版 — 快速扫描

```
@[/jinjian-perspective] NVDA 现在贵不贵？能买吗？
```

> 触发：§2.1 PE估值快扫 → PEG信号灯 → 买/卖/持有判断

### 进阶版 — 完整分析

```
@[/jinjian-perspective] 请用金渐成视角对 NVDA 做一次完整分析。
当前信息：股价 $178，VIX 22，市场情绪偏乐观。
我目前持仓占比 35%，成本 $120，已负成本。
```

> 触发：6步SOP全流程 → 实时数据拉取 → 历史锚点对比 → 仓位诊断 → 操作建议

### 估值对比版

```
@[/jinjian-perspective] NVDA 当前 Forward PE 是 28x，
对比 §7.5 估值锚点档案中的 24x，是不是偏贵了？
EPS 增速如果从 35% 降到 25%，PEG 会怎么变？
```

> 触发：§2.1 校准参数表 → PEG 敏感性分析 → 历史自比较

### 多股对比版

```
@[/jinjian-perspective] 现在手上有 $10万 现金准备建仓，
NVDA ($178) vs META ($620) vs MSFT ($400)，
从 PEG、FCF 和当前估值位置来看，优先买哪个？
```

> 触发：估值校准表横向对比 → 赛道逻辑 → 资金分配建议

### 深度报告版

```
@[/jinjian-perspective] 请给我出一份 GOOGL 的深度研判报告，
包含：趋势判断、估值快扫、历史操作回顾、未来买卖区间设置。
参考之前的 NVDA 深度报告格式。
```

> 触发：完整SOP + 搜索原始存档 + 决策时间线 + 输出完整报告

## 二、市场情景判断

### VIX 信号灯查询

```
@[/jinjian-perspective] VIX 现在到了 35，我该怎么操作？
目前现金占比 30%，持仓七巨头为主。
```

> 触发：§2.6 情景应对表 → VIX≥30 开始捞宽基 → 具体操作清单

### 暴跌应对

```
@[/jinjian-perspective] 市场突然暴跌 15%，VIX 飙到 45，
这跟 2025-04 关税战那次情况类似吗？
当时机哥怎么操作的？我现在该怎么做？
```

> 触发：情景应对 + 2025-04 实战案例交叉对比 + 金字塔加仓建议

### 泡沫检测

```
@[/jinjian-perspective] AI 板块最近涨了很多，
CapEx 还在暴增但云业务收入增速开始放缓，
这是不是泡沫破裂的信号？链条会断吗？
```

> 触发：§2.5 赛道验证（CapEx→收入链检查）→ AI链安全网评估

### 黑天鹅事件

```
@[/jinjian-perspective] [某公司] 突然爆出重大负面消息，股价暴跌 30%，
请帮我做一下粪坑检测四项，判断是"粪坑"还是"黄金坑"。
```

> 触发：Model 5 粪坑检测4项逐项打分 → UNH案例对比 → 操作建议

### 宏观事件影响

```
@[/jinjian-perspective] 美联储下周可能降息 25bp，
对我持仓的七巨头和美债ETF分别有什么影响？
该做什么调整吗？
```

> 触发：§2.6 美联储降息场景 → 成长股/债券分别影响 → 配置微调

## 三、仓位与配置诊断

### 组合体检

```
@[/jinjian-perspective] 帮我做个组合体检，目前持仓如下：
- NVDA 45%，成本 $120（负成本）
- MSFT 12%，成本 $380
- META 8%，成本 $550
- QQQ 15%
- 现金 20%
哪里有问题？需要调整吗？
```

> 触发：§2.4 仓位上限检测 + 三账户分类 + 集中度诊断 + 调整方案

### 负成本操作

```
@[/jinjian-perspective] 我的 NVDA 已经翻了 3 倍，
应该怎么做负成本？一次减多少？减出来的钱放哪里？
```

> 触发：Model 3 负成本 → TSM 7个月全程案例 → 2-3-3-2 反向协议 → 资金瀑布流

### 防守型账户设计

```
@[/jinjian-perspective] 我想建一个防守型账户，
预算 $20万，目标是"铠甲+安全垫"。
帮我设计具体配置和买入节点。
```

> 触发：Model 4 Tier 3 → BRK 时间线 → 美债ETF + SCHD + 高息股配比

### 三账户比例调整

```
@[/jinjian-perspective] 我现在进取型占了 65%、稳健型 15%、防守型 20%，
跟机哥说的目标比例 4:1.5:4.5 差距很大。
给我一个分阶段调整方案。
```

> 触发：三账户体系 → 踩坑案例（5.8:1.5:2.7 回撤大）→ 分步调整路径

### 新资金分配

```
@[/jinjian-perspective] 我刚到账 $5万新资金，现在市场处于震荡期（VIX 23），
应该怎么分配？一次性买入还是分批？买什么？
```

> 触发：VIX 信号灯（20-25 = 做T降成本）→ 资金分配建议 → 按三账户比例

## 四、赛道与标的筛选

### 赛道可行性

```

从金渐成的框架来看，这个赛道能碰吗？
```

> 触发：§2.5 赛道选择 → Model 5 粪坑检测 → 不碰清单对比

### 新标的评估

```
@[/jinjian-perspective] 我在考虑买入 COST（Costco），
这个标的符合金渐成的选股标准吗？
属于进攻型还是防守型？
```

> 触发：赛道分类（消费 = 稳健型）→ 估值快扫 → 负成本可行性判断

### 备胎策略

```
@[/jinjian-perspective] 如果 NVDA 有风险，
AMD 和 AVGO 作为备胎够不够？应该各配多少？
```

> 触发：§2.7 AMD+AVGO 时间线 → "一主两副"逻辑 → 合计占比控制

> 触发：§2.7 多只个股决策时间线 → 2025-04 集体抄底案例
