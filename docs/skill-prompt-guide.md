# SKILL v2.5.1 提问指南 — 如何高效调用“金渐成视角”

> **核心原则**：先说清你问的是“当前操作 / 年度展望 / 历史回顾 / 个人组合”，再补足背景，输出质量会明显提升。  
> 所有提问前建议加 `@[/jinjian-perspective]` 来激活技能模式。

---

## 调用方式说明

| 调用方式 | 含义 | 适用场景 |
|---------|------|---------|
| `@[/jinjian-perspective]` + 问题 | 激活技能：按最新模型、纪律、语气执行 | 投资分析、资产配置、宏观判断 |
| `@[.agent/skills/jinjian-perspective]` | 阅读技能文件本身 | 修改 / 升级 SKILL |

## 先记住这 4 条

1. **操作/估值类问题要先有新数据**
   例如“现在能买吗”“估值贵不贵”，默认要拉最新价格、估值、VIX、财报。
2. **先分清时间框架**
   `2026-04.md` 是战术档案，`Global Market Outlook for 2026.md` 是年度路线图，别把两个时框混成一个点位。
3. **历史观点要回源文件**
   现在仓库默认是 `source-first`：直接查 `22-25year/` 与 `26year/`，不再依赖月度索引。
4. **个人组合不是永久事实**
   如果你没给最新持仓，默认会参考 [personal-current-context.md](/Users/johnny/Documents/jjc-money/docs/personal-current-context.md)，但超过 14 天就要当成可能过期。

## 问题路由

| 模式 | 你在问什么 | 典型例子 |
|------|------------|---------|
| `Ticker` | 单只股票贵不贵、能不能买、怎么加减仓 | NVDA / MSFT / META |
| `Portfolio` | 三账户结构、仓位比例、资金流向 | 进取/稳健/防守怎么配 |
| `Personal Portfolio` | 结合你自己的持仓、现金、成本、目标来分析 | 我现在该卖谁、买谁 |
| `Macro Outlook` | 年度展望、跨资产、全球市场、汇率、大宗 | 2026 年怎么看铜、日本、港股 |
| `Historical View` | 作者当时怎么想、怎么做 | 2025-04 关税战时怎么接 |
| `Life Decision` | 顺势、能力圈、认知、职业/人生问题 | 跳槽、创业、买房 |

## 一、个股分析

### 基础版 — 快速估值扫描

```text
@[/jinjian-perspective] NVDA 现在贵不贵？能买吗？
```

适用：想快速触发 `PE / PEG / FCF` 快扫。

### 进阶版 — 当前操作分析

```text
@[/jinjian-perspective] 请用金渐成视角对 NVDA 做一次完整分析。
我问的是当前操作，不是历史回顾。
当前信息：股价 $178，VIX 22，市场情绪偏乐观。
我目前持仓占比 35%，成本 $120，已负成本。
```

适用：触发完整 `Ticker SOP`，会先取新数据，再对照历史锚点。

### 年度路线版

```text
@[/jinjian-perspective] 站在 2026 年全年视角，
NVDA 的年度支撑位、目标位和失效条件怎么设？
请优先参考 Global Market Outlook for 2026。
```

适用：明确要求走“年度展望”，避免混入 4 月战术挂单价。

### 历史回顾版

```text
@[/jinjian-perspective] 回顾一下 2026 年 4 月那轮下跌，
机哥为什么在那些点位接 NVDA？
我问的是历史逻辑，不是现在该不该买。
```

适用：触发“历史框架”模式，不强行给当前操作建议。

## 二、宏观与跨资产

```text
重点看：美股 / 美债 / 美元 / 铜铝 / 日本股市 / 英国富时 / 港股 / BTC。
```

适用：触发 `Macro Outlook` 路由，优先使用 [Global Market Outlook for 2026.md](/Users/johnny/Documents/jjc-money/26year/Global%20Market%20Outlook%20for%202026.md)。

### 战术节奏版

```text

叠加降息落空风险，我该怎么理解仓位节奏？
```

适用：触发“日历风险叠加 + 情景应对表”。

### 加密观察版

```text
@[/jinjian-perspective] 2026 年如果 BTC 回落，
你会在什么区间开始关注？这是主战场还是条件式参与？
```

适用：触发 `BTC 5.6万 / 5万 / 4.5万` 防守线逻辑。

## 三、资产配置与仓位管理

### 组合体检

```text
@[/jinjian-perspective] 帮我做个组合体检。
当前结构：
- 进取型：...
- 稳健型：...
- 防守型：...
- 现金：...
我更在意防守，不想大回撤。
```

适用：触发 `Portfolio` 路由，重点看三账户比例、资金流向、最大风险。

### 防守型账户设计

```text
@[/jinjian-perspective] 我想建一个防守型账户，
预算 $20万，目标是“铠甲+安全垫”。
请按 2026 年的思路，讲清楚美债、伯克希尔、高股息怎么分工。
```

适用：触发“伯克希尔防守引擎 + 现金像氧气”口径。

### 新资金配置

```text
@[/jinjian-perspective] 我刚到账 $5万新资金。
当前 VIX 23，想分配到进取/稳健/防守三账户里。
请给我框架，不要直接替我下单。
```

适用：触发“Advice Framing”，输出场景带，不给个性化指令。

## 四、个人组合

### 基础版

```text
@[/jinjian-perspective] 请基于我的个人组合来分析。
年龄：35
最大可承受回撤：-30%
未来半年无新增资金
当前持仓：...
现金：...
当前意图：想集中到 MSFT + NVDA
```

适用：触发 `Personal Portfolio` 路由。

### 使用仓库快照版

```text
@[/jinjian-perspective] 直接按 docs/personal-current-context.md 这份快照分析，
但请先判断它是不是已经过期。
```

适用：触发 `14天过期检查`，如果过期会先提醒。

### 交易卡片/计划协同版

```text
@[/jinjian-perspective] 结合我的 current context、portfolio card 和 long plan，
告诉我接下来应该先卖谁、谁只在回调时买。
```

适用：触发“最新用户输入 > current context > card > plan”的优先级。
