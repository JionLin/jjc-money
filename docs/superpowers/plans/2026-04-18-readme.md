```md
# jjc-money

一个按月份整理的 Markdown 内容归档，主要收录「金渐成」相关文章与相关内容。

这是一个长期维护的个人内容集，也是一份持续累积的写作归档。仓库中的内容主要按月份整理，适合按时间线浏览，也方便后续继续补充与维护。

## 仓库说明

这个仓库更偏向个人内容沉淀，而不是技术项目代码库。当前主要内容位于 `22-25year/`，以月度 Markdown 文件的形式保存，覆盖多个时间段的文章与互动内容。

## 目录结构

- `22-25year/`：按月份整理的 Markdown 归档内容
- `docs/indexes/`：索引与辅助文档
- `CLAUDE.md`：仓库协作与内容处理说明

## 内容格式

大多数月度文件通常包含以下结构：
- 月份标题
- 多篇文章条目
- 评论区或互动内容

不同时间段的文件格式并不完全一致。较早的文件更接近原始导出内容，而较新的整理文件会带有更完整的展示信息。

## 维护说明

- 新增内容时，尽量延续现有的月度归档方式
- 编辑已有内容时，尽量保持周边格式一致
- 查找文章时，优先通过日期或标题定位

## English

`jjc-money` is a personal Markdown archive of articles and related content, mainly organized by month.

Most of the archive lives in `22-25year/`, while `docs/indexes/` contains supporting indexes. If you are browsing this repository for the first time, start with the monthly files under `22-25year/`.
```
